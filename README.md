# US Weather App

This repository contains a single-file weather web app for US ZIP codes.

Update (2026-03-11): `index.html` was replaced with the previous `compare.html` content. The files `distance.html` and the original `compare.html` were removed.

Quick start:

Open `index.html` in a browser:

```bash
open index.html
```

If you keep changes in git, a suggested commit message is:

```bash
git add index.html
git rm distance.html compare.html
git commit -m "Replace index.html with compare; remove distance.html and old compare.html"
```

The app uses plain HTML, CSS and JavaScript — no build step required.
